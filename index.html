Smart ATS - Resume Evaluation Tool

Smart ATS is an application designed to evaluate resumes using the Large Language Models(LLM). By leveraging Google Gemini Pro Model, it thoroughly assesses resumes in comparison to provided job descriptions. This tool aims to aid job seekers in optimizing their resumes by identifying areas for improvement and ensuring alignment with job requirements.

This project can be run with streamlit. Instructions are given in the repository!
